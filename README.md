# LG Remote Mouse

Use a web browser to control the mouse on a WebOS TV. This project provides a lightweight Python server that lets you move the cursor and click on your LG WebOS television using any device with a browser.

# Quick start

 - Make sure to set the `IP` and `client_key` of your TV are set in `config.json`
 - Install dependencies: `pip install -r requirements.txt`
 - Run and connect your browser to the displayed address
